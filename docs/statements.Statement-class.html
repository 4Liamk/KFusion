Class Statement

source code

Statement Class: Stores a C statement. They are parsed recursively. Not all C statements are supported, switch cannot be used. It may not parse everything exactly as it should and takes a few shortcuts. It handles loops and scope particularly well though.

Can be parsed from file or if you pass in a type, it will return you a blank function of that type
